# brownian
Практическое доказательство закона Эйнштейна-Смолуховского о броуновском движении с помощью симуляции.
Код симулирует движение частиц в одномерной среде (количество частиц и шагов указывает пользователь), находит их среднюю тракторию и средний квадрат отклонения частиц, после чего на графике сравнивает полученные путем симуляции данные с предполагаемыми в теории. 
В основе модели лежит **закон больших чисел**: уравнение Ейнштейна <x**2> = t описывает поведение бесконечного числа частиц. Когда частиц мало, траектории не успевают усредниться и средний квадрат отклонения уходит от матожидания. На этом принципе работает вся статистическая физика и финансовая математика. Для корректного поведения системы рекомендуется количество частиц не менее 1000 и 500+ шагов.
